=== SISTEMA DE PLANEJAMENTO - SHAREPOINT ===

COMO USAR:

1. COPIAR PARA SHAREPOINT:
   - Crie uma pasta no SharePoint
   - Faça upload dos 3 arquivos principais:
     * index.html
     * style.css  
     * script.js

2. ACESSO:
   - Usuários com acesso à pasta poderão abrir o index.html
   - Funciona diretamente no navegador

3. FUNCIONALIDADES:
   ✅ Criar, editar e excluir tarefas
   ✅ Subtarefas com prazos individuais
   ✅ Prioridades (Alta, Média, Baixa)
   ✅ Status (Pendente, Em Andamento, Concluído)
   ✅ Filtros e busca
   ✅ Responsável por tarefa
   ✅ Prazos e alertas de atraso
   ✅ Exportar/importar dados
   ✅ Estatísticas em tempo real

4. MIGRAÇÃO FUTURA:
   - Os dados ficam no localStorage do navegador
   - Para migrar para servidor, basta substituir as funções de armazenamento
   - Estrutura de dados já preparada para APIs REST

5. DICAS:
   - Use "Exportar" para fazer backup regular
   - Responsáveis podem ser nomes ou departamentos
   - Subtarefas ajudam a quebrar tarefas complexas
